\section{Examples for the changes package}

Official documentation can be found at \url{http://mirrors.ibiblio.org/CTAN/macros/latex/contrib/changes/changes.english.pdf}.

Please use the following commands according to your change:

for added text:
\added[id=CdS]{This is a new text added}.
\added[id=CdS, comment=Including a comment]{This is another new text added}

for deleted text:
\deleted[id=CdS]{Text to be removed.}

for replaced text:
\replaced[id=CdS]{Inserting a new text}{Replacing this old text}.

Lorem ipsum dolor sit amet, consectetur adipiscing elit.
